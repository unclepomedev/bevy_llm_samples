# bevy_llm_samples

This repository hosts LLM-generated sample code (Claude, Gemini, etc.). Verified versions can be found in the Cargo.toml of each subdirectory. Please note that no guarantees or official support are provided.

* Keep all comments as-is, even the strange ones.
* Include the prompts where possible, but omit the back-and-forth struggles.
